# Moderasyon Botu
Profesyonel moderasyon ve yönetim botu. Ban, kick, mute, log ve rol sistemi ile sunucunuzu güvenli tutar.

## Dosyaları İndirme
Python ile yazılmış bot dosyalarını şu linkten indirebilirsiniz:  
[İndir](https://mega.nz/file/C1Rz2L4R#KiSmXHmow7g8_EcEZ3Ku4NFM_wmkrY1WijFMZzBhtsM)

## Kurulum
1. Python 3.11+ kurulu olmalı
2. Gerekli paketleri yükleyin: `pip install -r requirements.txt`
3. `.env` dosyasına bot tokeninizi ekleyin
4. `start.bat` ile botu başlatın

## Komutlar
- `/ban` - Kullanıcıyı sunucudan yasaklar
- `/kick` - Kullanıcıyı atar
- `/mute` - Kullanıcıyı susturur
- `/unmute` - Susturmayı kaldırır
- `/giverol` - Rol verir
- `/log` - Denetim kanalını hazırlar
- `/help` - Komutları gösterir

## Özellikler
- Forceban sistemi
- Loglama ve denetim
- Raid koruma (opsiyonel)
- 7/24 çalışabilir (bilgisayardan açarsanız)

## Not
Botu açmak için bilgisayarınızın sürekli açık olması gerekiyor. Mobil veya cloud sunucu kullanabilirsiniz.
